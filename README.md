## Pengaturan-Bot

1. Download and install [Node.js](https://nodejs.org) version `16.6.0`
2. Buka `.env_example` ubah nama menjadi `.env`
3. Instal dependensi yang diperlukan dan opsional
```sh
$ npm install
```
5. Anda dapat memulai bot
```sh
$ ts-node .
```
